# 易 — 線上問卦

> 靜心凝神，以易經六十四卦指引方向

**線上體驗：** [biology-c.github.io/I-Ching-reading](https://biology-c.github.io/I-Ching-reading/)

---

## 功能

- **三種起卦方式**
  - ⚡ 一念成卦 — 隨機起卦，心誠則靈
  - 🔢 數字起卦 — 輸入六組三位數，以數理推卦
  - 🪙 擲幣問天 — 模擬三枚銅錢，六次定爻

- **三段式解卦**
  - 現狀（本卦卦辭）
  - 轉折（動爻爻辭）
  - 趨向（變卦結語）

- **四大問卦方向** — 感情、學業、工作、財運

- **社交分享與互動**
  - 🔗 **分享專屬連結** — 複製帶有 hash 參數的網址，對方開啟即還原同一卦象
  - 🖼️ **生成結果圖片** — 點擊「儲存圖片」即可自動將排版轉為精美的分享卡片
  - ❤️ **喜歡與意見回饋** — 內建回饋星級評分機制與愛心點擊，紀錄您的真實感想
